Sobre Tab 3 Urna Eletrônica Aplicativo versão 1.0 Trabalho feito para cadeira de Desenvolvimento Mobile da turma de Engenharia da computação do 7ª período.
O intuito do aplicativo foi para poder viabilizar a votação de qualquer que seja o local que eteja, sem precisar sair da sua comodidade. Para realização do projeto utilizamos o Ionic com integração Cordova, a partir das tags do Ionic fomos montando todo o corpo do projeto.
